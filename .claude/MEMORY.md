## Remote Script Installation Path

The Ableton Remote Script is installed at:
```
~/Music/Ableton/User Library/Remote Scripts/AbletonMCP_Remote_Script/__init__.py
```

When you modify `AbletonMCP_Remote_Script/__init__.py` in the project repo, copy it to this path:
```bash
cp AbletonMCP_Remote_Script/__init__.py "/Users/noamisraeli/Music/Ableton/User Library/Remote Scripts/AbletonMCP_Remote_Script/__init__.py"
```

After copying, reload the Remote Script in Ableton:
1. Go to **Preferences → Link, Tempo & MIDI**
2. Set Control Surface to **None**, then back to **AbletonMCP_Remote_Script**
